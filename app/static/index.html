Jsem progamátor na volné noze z Ostravy

Specializuji se především na programování v jazyce Python, tedy backend, ale poradím si i s jazyky a technologiemi příbuznými.

Momentálně se věnuji práci na zakázku a soustředím se na regionální zákazníky.

Pokud byste potřebovali pomoct s něčím z oblasti programování, podívejte se, co nabízím a jestli jsem pro Vás ten pravý.

Co nabízím

Poskytuji různé druhy celých aplikací, nebo i dílčí úpravy, případně knihovny či moduly do Vašeho stávajícího kódu.

Multiplatformní aplikace

Mohu poskytnout aplikace běžící pod Windows, Linuxem či macOS.

Stejně tak je možné napsat aplikace pro osobní počítače, servery, nebo embedded zařízení.

S uživatelským rozhraním

Grafické uživatelské rozhraní (GUI) - desktopové a nebo vizuálně jednoduché webové aplikace.

Konzolové aplikace (CLI) - aplikace s terminálovým rozhraním.

S aplikačním rozhraním

Standardní HTTP API typu REST, GraphQL, RPC, případně přes socket.

Kód, knihovny, moduly

Krom celých aplikací mohu poskytnout i dílčí úpravy kódu, knihovny nebo moduly.

Hlavně v jazyce Python.

Provoz

Aplikace může být provozována u Vás a nebo mohu provoz zajistit. Záleží však na typu a náročnosti aplikace.

Dobré zkušenosti mám např. s Roští.cz.

Podpora

Pokud máte se svou aplikací problémy, můžete se na mě obrátit i s naléhavými požadavky.

Pokud budu moct, rád pomohu.

Různé funkce

Napsané aplikace či kód mohou plnit různé funkce dle Vašich požadavků. Často se jedná o databázové systémy, automatizační úlohy, zpracování dat, obrazu, generování dokumentů (např. PDF, CSV, HTML, XML), práci s e-maily, jednoduchý web, API, nebo práci s API třetích stran (např. sociálních sítí), ale i další.

Neváhejte mě v případě zájmu kontaktovat.

S čím nepomohu

Vizuálně náročné weby

Prezentační vrstva je doménou frontendových programátorů. Já jsem především backend programátor, proto poskytuji především backendové aplikace a nebo vizuálně příliš nenáročné aplikace.

Grafický návrh

Mohu nabídnout graficky zajímavý web postavený např. na šabloně jako je HTML5 UP. Sám se ale grafikou příliš nezabývám a grafický návrh na míru neposkytuji.

Mobilní aplikace

Programuji především v jazyce Python a ten není na mobilní aplikace příliš vhodný. Proto se vývojem mobilních aplikací nezabývám.

Hry

Stejně jako není Python příliš vhodný pro mobilní aplikace, není ani přliš vhodný pro programování her. Proto se vývojem her nezabývám.

24/7 podpora a kritický software

Pracuji sám na sebe a nemám jako samostatný vývojář zastupitelnost, nemohu proto zajistit neustálou podporu, např. v případě nemoci, dovolené atp. Proto neposkytuji software, který je pro klienta kritický, pokud nemá alespoň částečně zajištěnu podporu na své straně.
